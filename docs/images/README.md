## üìÅ Where to Upload Your Images

Place all your images and files here: `/public/images/`

### Current Image Requirements:

1. **`students-hero.jpg`** - Hero section background (Homepage)
   - Recommended size: 1920x500px
   - Format: JPG, PNG, or WebP
   - Shows students/school building

2. **`back-to-school.jpg`** - News article image
   - Recommended size: 800x400px
   - Format: JPG, PNG, or WebP

### How to Add New Images:

1. Copy your image files to this folder
2. Reference them in your code using: `/images/your-file-name.jpg`
3. For news articles, add `"imageUrl": "/images/your-image.jpg"` to the news item in `src/data/news.json`

### Supported Formats:
- JPG/JPEG
- PNG
- WebP
- SVG
- GIF

### File Size Recommendations:
- Hero images: < 500KB
- News images: < 200KB
- Icons: < 50KB

Use online tools like [TinyPNG](https://tinypng.com/) to compress images before uploading.
